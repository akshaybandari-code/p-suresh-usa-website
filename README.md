# SuureshUSA – Professional Website & Admin CMS

A modern, responsive business website and Content Management System (CMS) developed for **P. Suresh & Associates**, a Chartered Accountancy firm providing cross-border taxation and financial consultancy services for clients in the United States.

The project includes a public-facing website for clients and a secure Admin CMS for managing website content such as services, articles, tax updates, downloadable resources, and team members.

# Installation

Clone the repository

```bash
git clone https://github.com/akshaybandari-code/p-suresh-usa-website.git
```

Go to the project directory

```bash
cd p-suresh-usa-website
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

# Build for Production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```

---

# Environment Variables

Create a `.env` file using `.env.example`.

Example:

```env
VITE_EMAILJS_SERVICE_ID=YOUR_SERVICE_ID
VITE_EMAILJS_TEMPLATE_ID=YOUR_TEMPLATE_ID
VITE_EMAILJS_AUTOREPLY_TEMPLATE_ID=YOUR_TEMPLATE_ID
VITE_EMAILJS_PUBLIC_KEY=YOUR_PUBLIC_KEY
```
